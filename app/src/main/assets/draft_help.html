Draft Scout tries to evaluate the data (all matches so far) saved by scouting and produce a "ranking" of teams for Draft Alliance selection using the following calculations.   You can vary/change the calculation by changing the modifiers for each item by using the 'Settings' feature in the Options Menu ( ⁝ ) in the Tool Bar at the top right.   These multipliers are represented by UPPER case letters (or Greek symbols) in the formulas below.

Power Cells

Scored
((B*(autoCellLow) + C*(autoCellUnder)) +D*(autoCellUnder) + E*(autoCellLine) + F*(autoCellFront) + (G*(teleCellLow) + H*(teleCellUnder)) + I*(teleCellUnder) + J*(teleCellLine) + K*(teleCellFront)+ L*(teleCellBack)) / # Matches
  • A - Multiplier
  • dump - number of time Cells dumped to another robots during Autonomous
  • B - Multiplier
  • autoCellLow - number of Cells scored in Power Port (PP) Low goal during Autonomous
  • C - Multiplier
  • autoCellUnder - number of Cells scored on Sector Line during Autonomous
  • D - Multiplier
  • autoCellLine - number of Cells scored on Sector Line during Autonomous
  • E - Multiplier
  • autoCellFront - number of Cells scored from in front of Control Panel (C.P.) during Autonomous
  • F - Multiplier
  • autoCellFront - number of Cells scored from in front of C.P. during Autonomous
  • G - Multiplier
  • teleCellLow - number of Cells scored in Power Port (PP) Low goal during TeleOps
  • H - Multiplier
  • teleCellUnder - number of Cells scored on Sector Line during TeleOps
  • I - Multiplier
  • teleCellLine - number of Cells scored on Sector Line during TeleOps
  • J - Multiplier
  • teleCellFront - number of Cells scored from in front of C.P. during TeleOps
  • K - Multiplier
  • teleCellBack - number of Cells scored from in back of C.P. during TeleOps
Collected
( M*(autoFloor) + N*(autoCP) + O*(autoTrench) + P*(autoBoundary) + Q*(autoRobot) + ( R*(teleFloor) + S*(teleCP) + T*(teleTrench) + U*(teleBoundary) + V*(teleRobot) + W*(teleLoadSta)) / # Matches
  • autoFloor - collected Cells retrieved from Floor during Autonomous
  • autoCP - collected Cells retrieved from C.P. during Autonomous
  • autoTrench - collected Cells retrieved from Trench during Autonomous
  • autoBoundary - collected Cells retrieved from SG Boundary during Autonomous
  • autoRobot - collected Cells retrieved from another Robot during Autonomous
  • teleFloor - collected Cells retrieved from Floor during TeleOps
  • teleCP - collected Cells retrieved from C.P. during TeleOps
  • teleTrench - collected Cells retrieved from Trench during TeleOps
  • teleBoundary - collected Cells retrieved from SG Boundary during TeleOps
  • teleRobot - collected Cells retrieved from another Robot during TeleOps
  • teleLoadSta - collected Cells retrieved from another Robot during TeleOps
Cell Score is the sum of of Scored and Collected

Control Panel (C.P.)

((X*(CPspin) + Y*(CPcolor)) / # Matches
  • CPspin - Spin C.P. number of revolutions
  • CPcolor - Spin & Stop C.P. on correct color

Climb

(α*(climbs) + β*(Lift1) + γ*(Lift2) + δ*(undSG) + ε*(WasLifted)) / # Matches
  • climbs - number of successful climbs by this robot
  • Lift1 - number of times this robot lifted one robot
  • Lift2 - number of times this robot lifted two robots
  • undSG - number of times this robot was under the Shield Generator at end
  • WasLifted - number of times this robot was lifted by another team

Combined

( ζ*(climbScore) + η*(CellsScored) + θ*(CP)) / # Matches
  • climbScore - total from the Climb score
  • CellsScore - total from the Cells score
  • CP - total from the C.P. score
